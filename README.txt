VISUAL TEST AUTOMATION BUILDER (VTAB)
Version 1.0.0
==================================================

Thank you for downloading VTAB!

INSTALLATION
-----------
1. Run the installer (VTAB_Setup_v1.0.0.exe)
2. Follow the installation wizard
3. Launch VTAB from Start Menu or Desktop shortcut

SYSTEM REQUIREMENTS
------------------
- Windows 10 or Windows 11 (64-bit)
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space
- 1280x720 minimum screen resolution

FIRST TIME SETUP
----------------
1. Launch VTAB
2. Create a new project or open an existing one
3. Drag nodes from the Node Library onto the canvas
4. Connect nodes to create your test flow
5. Click Execute to run your test

FEATURES
--------
- Visual node-based test creation
- Computer vision element detection
- AI-powered text verification (GPT-4o Vision)
- Real-time execution with debugging
- Professional HTML reports
- Dark theme interface
